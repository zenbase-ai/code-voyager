# /factory

Invoke the Skill Factory to propose or scaffold new skills.

## What You Can Do

1. **Propose skills** — Analyze patterns and suggest new skills
2. **Scaffold a skill** — Create a skill folder from a proposal
3. **List proposals** — See available skill candidates

## Examples

- "Propose some skills based on my recent work"
- "What skills could I create?"
- "Scaffold the pr-reviewer skill"
- "Create a skill for our deployment process"
- "List the proposed skills"

Use the skill-factory skill for implementation details.
